## How to run project.

### Development 

- backend

  `cd backend`

  `npm i`

  `npm run dev`
  
  or

  `yarn dev`

- frontend

  `cd frontend`

  `npm i`

  `yarn start`

### Production

- backend

  `npm run start`

- frontend

  `npm run build`
